# 快捷键参考

`__or_preview.js` 为所有 HTML 目录页注入了键盘导航与媒体预览功能，快捷键分为两个上下文：

- **目录列表模式**：预览叠加层关闭时，焦点框在目录条目间移动
- **预览模式**：媒体预览叠加层打开时，操控媒体内容

---

## 目录列表模式

| 键位 | 功能 |
|------|------|
| `↑` | 焦点移到上一个条目 |
| `↓` | 焦点移到下一个条目 |
| `Enter` | 打开/激活当前焦点条目（媒体文件→打开预览叠加层；目录→进入该目录（同页跳转）；其他文件→跟随链接） |
| `Del` | 删除当前焦点条目（文件或目录，目录会二次确认并提示"所有内容将一并删除"） |
| `B` | 返回上一级目录，返回后焦点自动定位到刚才进入的那个目录条目 |

### 焦点恢复（B 键返回）

进入子目录前，当前子目录的路径会存入 `sessionStorage['__or_back_from']`。  
返回父页面时，`init()` 读取该值，在链接列表中找到对应目录项，自动滚动并恢复焦点框。

---

## 预览模式（媒体叠加层打开时）

### 导航

| 键位 | 功能 |
|------|------|
| `↑` / `←` | 上一个媒体文件（图片模式下左右键等同上下键；到头不循环） |
| `↓` / `→` | 下一个媒体文件（图片模式下左右键等同上下键；到尾不循环） |
| `Home` | 跳到第一个媒体文件 |
| `End` | 跳到最后一个媒体文件 |
| `Esc` / `B` | 关闭预览，回到目录列表，恢复焦点框 |

> **注意**：`←` / `→` 在视频/音频状态下会改为跳帧功能（见下方），图片状态下仍为前/后翻页。

### 视频 / 音频控制

| 键位 | 功能 |
|------|------|
| `Space` | 播放 / 暂停 |
| `M` | 静音 / 取消静音（OSD 显示 🔇 / 🔊） |
| `←` | 快退 5 秒 |
| `→` | 快进 5 秒 |
| `Ctrl + ←` | 快退 30 秒 |
| `Ctrl + →` | 快进 30 秒 |

**触屏：** 在视频画面上**长按**（≥300ms）激活 2× 快进，松开恢复原速。

**播放速度下拉：** 工具栏右上角 `<select>` 可选 0.5× / 1× / 1.5× / 2× / 4× / 10×，仅视频/音频时显示。

### 视图模式（图片 / 视频）

| 键位 | 工具栏按钮 | 功能 |
|------|-----------|------|
| `F` | ⛶ | 浏览器**全屏**切换（全屏时工具栏自动隐藏，hover 时显现） |
| `O` | 1:1 | 切换 **适应窗口 (Fit)** ↔ **原始大小 (Orig)**（1:1 像素，可滚动） |
| `V` | 🔄 | 切换 **适应窗口 (Fit)** ↔ **长边旋转 (Rotate)**（自动旋转使长边对齐屏幕长边，方便竖版图横屏观看） |

三种视图模式：

| 模式 | 说明 |
|------|------|
| `fit`（默认） | 等比缩放到窗口大小，不裁切，长宽都显示完整 |
| `orig` | 以原始像素尺寸展示，超出视口时可滚动查看 |
| `rotate` | 自动旋转 90°，使图片/视频的长边对齐屏幕长边（竖版内容横屏看）|

### 删除

| 键位 | 功能 |
|------|------|
| `Del` | 删除当前预览的媒体文件（弹框确认，删除后自动跳到下一个） |

> 压缩包（ZIP/CBZ）内部文件不支持删除，`Del` 键此时不可用（按钮变灰）。

---

## 鼠标 / 触控

| 操作 | 功能 |
|------|------|
| 点击预览图标（🖼️ / ▶️ / 🎵） | 打开对应文件的媒体预览叠加层 |
| 点击媒体文件的**文字链接** | 在新标签页打开文件（不进入预览叠加层） |
| 点击目录 / 非媒体文件链接 | 同页跳转（正常导航） |
| 点击预览图片区域 | 跳到下一个媒体文件（到尾不循环） |
| 点击叠加层背景（黑色区域） | 关闭预览 |
| 视频画面长按（触屏） | 激活 2× 快进，松开恢复原速 |
